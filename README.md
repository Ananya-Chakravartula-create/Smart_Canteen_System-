# Smart Canteen System 

##  Project Overview

**Smart Canteen System** is a web-based application designed to digitize and simplify the food ordering process in an office cafeteria.
It allows employees to register, log in, view the menu, place food orders online, and enables administrators to monitor all orders efficiently.

The system reduces waiting time, avoids manual order handling, and improves overall canteen management.



##  Objectives

* To provide a **digital food pre-ordering platform**
* To reduce queues and manual errors in cafeterias
* To maintain **user registration and order records**
* To provide an **admin dashboard** for order monitoring



##  Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** JSP (Java Server Pages), JDBC
* **Database:** Apache Derby (Java DB)
* **IDE:** NetBeans
* **Server:** Apache Tomcat



##  User Roles

###  Employee (User)

* Register and create an account
* Login using username and password
* View food menu 
* Add items to order
* Place food orders online

###  Admin

* Login using admin credentials
* View all placed orders
* Monitor order details such as user, item, price, and date



##  Database Design

### REGISTER Table

Stores user registration and login details:

* FULLNAME
* USERNAME
* PASSWORD
* MOBILE
* EMAIL

### ORDERS Table

Stores food order details:

* ORDER_ID 
* FULLNAME
* USERNAME
* ITEM
* PRICE
* ORDER_DATE



##  Features

* Secure login using database validation
* Session management for users
* Auto-generated order IDs
* Separate admin and user modules
* Simple and user-friendly interface
